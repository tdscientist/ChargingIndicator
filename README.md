## ChargingIndicator ##

*iOS charging indicator inspired by [Charging Animation](https://ios.uplabs.com/posts/charging-animation)*


<img src="/screenshot.gif" width="230">

</br>
#### Requirements ####
----
* xCode 8.2.1
* Siwft 3
* iPhone device running on iOS 8 and above
